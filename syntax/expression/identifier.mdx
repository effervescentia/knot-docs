# Identifiers

[Variables](../variable) created with a `const` or `let` bindings can be referenced by name.

<Tabs>
<TabItem value="knot" label="Knot">

```knot
const secretNumber = 123;
const moreSecretNumber = secretNumber + 1;
```

</TabItem>
<TabItem value="tsx" label="TypeScript">

```tsx
const secretNumber = 123;
const moreSecretNumber = secretNumber + 1;
```

</TabItem>
</Tabs>
