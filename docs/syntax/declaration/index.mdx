# Declarations

All declarations in a [module](../module) are available for [import](../module/import-export) from within other modules.

## [Constant](constant)

```knot
const TIMEOUT = 100;
const ERR_MESSAGE = "failed to connect";
const DEBUG = false;
```

## [Enumerated](enum)

```knot
enum Coordinates =
  | Point2D(float, float)
  | Point3D(float, float, float);
```

## [Function](function)

```knot
func add(lhs: integer, rhs: integer) -> lhs + rhs;
```

## [View](view)

```knot
view Header ->
  <header>
    <h1>Welcome!</h1>
  </header>;
```
