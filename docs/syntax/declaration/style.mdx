# Styles

Styles can be used to decorate [elements](../element) to describe how they should be rendered.
In a browser environment this is equivalent to styling elements using CSS.

## Definition

```knot
style CoreStyle { }
```

## Classes

Most styling will be done with classes.

<Tabs>
<TabItem value="knot" label="Knot">

```knot
style CoreStyle {
  .header {
    color: $red;
  }
}
```

</TabItem>
<TabItem value="css" label="CSS">

```css
.header {
  color: red;
}
```

</TabItem>
</Tabs>

## Identifiers

You can also use identifiers for styling.

<Tabs>
<TabItem value="knot" label="Knot">

```knot
style CoreStyle {
  #hero {
    color: $red;
  }
}
```

</TabItem>
<TabItem value="css" label="CSS">

```css
#hero {
  color: red;
}
```

</TabItem>
</Tabs>
