# Statements

Closures in Knot can be empty or contain one or more statements.

## [Expression](../expression)

Any expression is a valid statement.

```knot
func foo -> {
  10 + 0.4;

  true && false;

  "fizz" == "buzz";
};

```

## [Let Binding](statement/let-binding)

If a `let` binding is the last statement in the closure then the [return value is implicitly `nil`](primitive/nil#implicit-return).

```knot
func foo -> {
  let bar = 10;

  let fizz = bar - 3 > 10;

  let buzz = !fizz;
};

```
