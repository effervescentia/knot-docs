# Statements

Closures in Knot can be empty or contain one or more statements.

## [Expression](../expression)

Any expression is a valid statement.

```knot
func foo -> {
  10 + 0.4;

  true && false;

  "fizz" == "buzz";
};

```

## [Let Binding](let-binding)

```knot
func foo -> {
  let bar = 10;

  let fizz = bar - 3 > 10;

  let buzz = !fizz;
};

```
