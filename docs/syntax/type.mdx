# Types

There are 6 built-in types that can be combined in type expressions to describe more complex types.

- [`nil`](primitive/nil)
- [`boolean`](primitive/boolean)
- [`integer`](primitive/number#integer)
- [`float`](primitive/number#float)
- [`string`](primitive/string)
- [`element`](element)
