# Logical Operators

These operations can be performed on [boolean](../primitive/boolean) values.

## Unary Operators

### NOT (`!`)

Used to invert a boolean value; `true` values become `false` and `false` values become `true`.

<Tabs>
<TabItem value="knot" label="Knot">

```knot
const foo = true;
const bar = false;

!foo == false;
!false == true;
!(foo && bar) == true;
```

</TabItem>
<TabItem value="tsx" label="TypeScript">

```tsx
const foo = true;
const bar = false;

!foo === false;
!false === true;
!(foo && bar) === true;
```

</TabItem>
</Tabs>

## Binary Operators

### AND (`&&`)

This operation evaluates to `true` if both the left-hand and right-hand expressions evaluate to `true` and `false` otherwise.

<Tabs>
<TabItem value="knot" label="Knot">

```knot
const foo = true;
const bar = false;

(foo && true) == true;
(bar && true) == false;
(bar && bar) == false;
```

</TabItem>
<TabItem value="tsx" label="TypeScript">

```tsx
const foo = true;
const bar = false;

(foo && true) === true;
(bar && true) === false;
(bar && bar) === false;
```

</TabItem>
</Tabs>

### OR (`||`)

This operation evaluates to `true` if either the left-hand or right-hand expressions evaluate to `true` and `false` otherwise.

<Tabs>
<TabItem value="knot" label="Knot">

```knot
const foo = true;
const bar = false;

(foo || true) == true;
(bar || true) == true;
(bar || bar) == false;
```

</TabItem>
<TabItem value="tsx" label="TypeScript">

```tsx
const foo = true;
const bar = false;

(foo || true) === true;
(bar || true) === true;
(bar || bar) === false;
```

</TabItem>
</Tabs>
